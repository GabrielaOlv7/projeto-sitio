o que é o HTML e CSS

HTML é uma linguagem de computador que compõe a maior parte das páginas da internet por ue dos aplicativos online.
hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta
por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código
criado por linguagens como HTML, XML ou XHTML, por exemplo.
De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.
